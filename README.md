# FathFath Next.js Demo

This is a demo Next.js + Tailwind project (UI-only) created for the user.
It replicates a polished panel UI (no game automation / no cheating functionality).

## Quick start

1. Install dependencies:
   ```
   npm install
   ```

2. Run dev:
   ```
   npm run dev
   ```

3. Open http://localhost:3000

## Deploy to Vercel

1. Push this repo to GitHub.
2. Import project at https://vercel.com (sign in with GitHub).
3. Vercel will detect Next.js and deploy automatically.
